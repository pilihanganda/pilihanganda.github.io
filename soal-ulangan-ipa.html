31. Nilai soal ulangan IPA dari beberapa siswa tercatat sebagai berikut : 77, 80, 67, 73, 73, 62, 75, 75, 88,70 Rata-rata nilai ulangan IPA tersebut adalah ..​


Jawaban:

Rata rata = jumlah data / banyak data

jumlah data :

= 77 + 80 + 67 + 73 + 73 + 62 + 75 + 75 + 88 + 70

= 740

banyak data = 10

rata rata = 740 / 10

[tex] \frac{740}{10} = 74[/tex]

jadi, nilai rata rata ulangan IPA tersebut adalah 74

36. Soal IPA Kelas IXButuh Cepat Buat Ulangan​


LISTRIK DINAMIS

▶ Rangkaian Resistor

Perhatikan gambar di bawah ini :

Besar hambatan total adalah .. ohm

R₁ = R₂ = R = 8 Ω

R₃ = 4 Ω

Hambatan pengganti untuk R₁ & R₂

Rp = R / n

Rp = 8 / 2

Rp = 4 Ω

Hambatan total

Rs = Rp + R₃

Rs = 4 + 4

Rs = 8 Ω ✔
