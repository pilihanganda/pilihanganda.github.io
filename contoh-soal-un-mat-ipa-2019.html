Matematika SMA Kelas X

lim ((3x - 5) / (2x² + 4x + 5))
x→∞

= lim ((3x/x² - 5/x²) / (2x²/x + 4x/x² + 5/x²))
  x→∞

= lim ((3/∞ - 5/∞²) / (2 + 4/∞ + 5/∞²)
  x→∞

= (0 - 0)/(2 + 0 + 0)
= 0/2
= 0
