1. Materi persamaan dan fungsi kuadrat dalam matematika meliputi pengertian, bentuk umum, sifat-sifat, metode penyelesaian, serta aplikasinya dalam kehidupan sehari-hari. Persamaan kuadrat adalah persamaan yang memiliki bentuk a x^2 + bx + c = 0, dengan a, b, dan c adalah bilangan real dan a tidak sama dengan 0. Sedangkan fungsi kuadrat adalah fungsi yang memiliki bentuk f(x) = a x^2 + bx + c, dengan a, b, dan c juga adalah bilangan real dan a tidak sama dengan 0.

2. Contoh Soal

Soal 1:

Diketahui persamaan kuadrat x^2 + 5x - 6 = 0, carilah akar-akarnya!

Jawab: x1 = (-b + √(b^2-4ac))/(2a) = (-5 + √(5^2-4(1)(-6)))/(2(1)) = (-5 + √(25+24))/2 = (-5 + √49)/2 = (-5 + 7)/2 = 2 dan x2 = (-5 - 7)/2 = -6

Soal 2:

Diketahui fungsi kuadrat f(x) = x^2 - 8x + 15, carilah nilai minimum dari fungsi tersebut!

Jawab: f(x) = a(x-b)^2 + c, dengan a = 1, b = -8/2 = -4, c = 15.

Jadi, nilai minimum dari fungsi tersebut adalah c = 15.
