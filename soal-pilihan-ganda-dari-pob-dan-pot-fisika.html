L tutup = ___ ?

Jawab :
Cek dulu frekuensi nada ke brp yg dihasilkan POB ....gunakan rumus :
P = 2 + n
S = 1 + n
Keterangan : n itu menunjukkan nada atas ke - n

Krn di POB terjadi 4 perut dan 3 simpul maka nilai n = ___?
P = 2 + n = 4 ==> n = 2
S = 1 + n = 3 ==> n = 2
Maka terjadi nada atas ke DUA pada POB
Perbandingan frekuensi pada POB
fo : f1 : f2 : f3 : ...... =
1. : 2 : 3. : 4 : ....

Maka f2 buka = 3.fo buka = 3.V/(2.L buka)


Cek dulu frekuensi nada ke brp yg dihasilkan POT ....gunakan rumus :
P = S = 1 + n

Keterangan : n itu menunjukkan nada atas ke - n

Krn di POT terjadi 3 perut dan 3 simpul maka nilai n = ___?
P = S = 1 + n = 3 ==> n = 2

Maka terjadi nada atas ke DUA pada POT
Perbandingan frekuensi pada POT
fo : f1 : f2 : f3 : ...... =
1. : 3 : 5. : 7 : ....

Maka f2 tutup = 5.fo tutup = 5.V/(4.L tutup)

Krn terjadi resonansi maka
f2 buka = f2 tutup
3.V/(2.L buka) = 5.V/(4.L tutup)

3.V/(2.96) = 5.V/(4.L tutup)

3/(2.96) = 5/(4. L tutup)

1/(2.32) = 5/(4. L tutup)

1/(64) = 5/(4. L tutup)

L tutup = (5/4) x 64

L tutup = 5 x 16

L tutup = 80 cm

7. jika luas juring AOB = 50 cm²,tentukan luas juring POB! ​


Luas juring POB adalah 131,8 cm².

Pembahasan

Lingkaran merupakan bangun datar yang tersusun dari beberapa titik yang memiliki jarak yang sama terhadap titik pusat, dimana jarak antara titik pusat dengan salah satu tutuk disebut jari-jari lingkaran.

Juring adalah daerah yang dibatasi busur dengan dua buah jari-jari.

Pelajari Lebih Lanjut Bab lingkaran Mencari jari-jari yang diketahui luas dapat disimak di brainly.co.id/tugas/1449306

Rumus-rumus bab lingkaran

Luas lingkaran = π x r² atau ¹/₄ x π x d²

r = [tex]\sqrt{\frac{L}{\pi } }[/tex]

d = [tex]\sqrt{\frac{4.L}{\pi } }[/tex]

Keliling lingkaran = 2 x π x r atau π x d

r = [tex]\frac{K}{2.\pi }[/tex]

d = [tex]\frac{K}{\pi }[/tex]

d = 2 x r

r = jari-jari lingkaran

d = diameter lingkaran

π =  atau 3,14

Luas Juring = [tex]\frac{Sudut~juring}{360}[/tex] x Luas lingkaran

Panjang busur = [tex]\frac{Sudut~juring}{360}[/tex] x Keliling lingkaran

Dapat juga menggunakan rumus

[tex]\frac{Sudut~A}{Sudut~B} =\frac{Luas~Juring~A}{Luas~juring~B} =\frac{Panjang~busur~A}{Panjang~Busur~B}[/tex]

Penyelesaian Soal

Diketahui:

Luas juring AOB = 50 cm²

Sudut AOB = 33°

Sudut POQ = 60°

Ditanya:

Luas juring POB

Jawab:

Langkah pertama, kita cari besar sudut BOP

Besar sudut BOP = 180 - (33 + 60)

                             = 180 - 93

                             = 87°

Langkah selanjutnya kita cari luas juring POB, dengan rumus perbandingan.

[tex]\frac{Sudut~AOB}{Sudut~BOP} =\frac{Luas~Juring~AOB}{Luas~Juring~POB}[/tex]

[tex]\frac{33}{87} =\frac{50}{y}[/tex]

33y = 87 x 50

y = 87 x 50 : 33

y = 131,8

Luas juring POB adalah 131,8 cm².
