= 7 mod 7
= 0

= 129 mod 7
= 0

5^205 + 2^205 mod 7 = 0

40. 5. Banyak suku pada barisan 9, 13, 17,21, ... , 205 adalah ....A. 50C. 55B. 51D. 60AP UN Matematika SMP/MTS​


Jawaban:

[ barisan deret aritmatika ]

9, 13, 17, 21, ...., 205.

a = suku Pertama = 9

b = beda suku = u² - a = 13 - 9 = 4

Un = 205

Un = a + (n - 1)b

205 = 9 + (n - 1)4

205 = 9 + 4n - 4

205 = 5 + 4n

4n = 205 - 5

4n = 200

n = 200/4

n = 50

Jawaban [ A ]
