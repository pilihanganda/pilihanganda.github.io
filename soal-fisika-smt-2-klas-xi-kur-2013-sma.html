4. Quiz Fisika Materi : Impuls Sudut XI SMA


Impuls Sudut.

Dari Hukum II Newton.
F = ma
F = mαr
Fr = mαr²
τ = Iα
τ = I(ω - ω₀) / Δt
τ Δt = ΔL → τ = ΔL / Δt
Terbukti. a adalah percepatan tangensial.

τ Δt = I Δω
τ Δt = mr² Δω
Terbukti.Nomor 1
-------------
Pembuktian rumus impuls pada gerak translasi menghasilkan impuls pada gerak rotasi :

F.Δt = ΔP
m.a.Δt = m.(V - Vo) .... pers (1)

Krn hubungan percepatan linier pada gerak translasi dan percepatan anguler pada gerak rotasi adalah
a = α.r

dan

Hubungan kecepatan linier dan kecepatan sudut adalah
V = ω.r

Maka persamaan (1) di atas berubah menjadi :
m.a.Δt = m.(V - Vo)
m.α.r .Δt = m.(ω.r - ωo.r)
m.α.r .Δt = m.r.(ω - ωo)
Kemudian semua ruas dikalikan r, menjadi :
 
m.r².α.Δt = m.r².(ω - ωo)

Dan karena Momen inersia partikel adalah, I = m.r²
Maka :
I.α.Δt = I.(ω - ωo)

Dan karena momen gaya atau Torsi adalah, τ = I.α

Maka rumus di atas menjadi :
I.α.Δt = I.(ω - ωo)
τ.Δt = I.ω - I.ωo

Dan karena momentum sudut adalah L = I.ω

Maka rumus di atas menjadi :
τ.Δt = I.ω - I.ωo
τ.Δt = L - Lo

τ.Δt = ΔL

τ = ΔL / Δt
(TERBUKTI)


Nomor 2
------------
Pembuktian bahwa impuls sudut partikel adalah :
τ.Δt = m.r².Δω

Bukti :
τ.Δt = I.ω - I.ωo
τ.Δt = I.(ω - ωo)
τ.Δt = I.Δω

Untuk partikel besar momen inersia nya adalah I = m.r²

Maka rumua Di atas menjadi :
τ.Δt = I.Δω
τ.Δt = m.r².Δω
(TERBUKTI)

6. Quiz Fisika Kelas XI SMA


jawaban terlampir



Semoga bermanfaat~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Diket :
F1 = - 5 N (Arah gaya ke BAWAH menyebabkan batang berputar MELAWAN jarum jam terhadap poros C)
F2 = 0,4 N (Arah gaya ke ATAS menyebabkan batang berputar SEARAH jarum jam terhadap poros C)
F3 = 4,8 N (Arah gaya ke BAWAH menyebabkan batang berputar SEARAH jarum jam terhadap poros C)

Panjang Batang 4 meter dibagi 6 ruas, maka masing-masing ruasnya sebesar, 1 ruas = 4/6 meter

L1 = 2 ruas = 2 x 4/6 m = 8/6 meter (di sebelah kiri titik poros C)
sudut, θ1 = 53°
L2 = 1 ruas = 4/6 meter (di sebelah kiri titik poros C)
sudut, θ2 = 90°
L3 = 2 ruas = 2 x 4/6 m = 8/6 meter (di sebelah kanan titik poros C)
sudut, θ3 = 90°

Tanya : 
Momen gaya, Torsi, τc = __?

Jawab :

τc = F1.L1.sin θ1 + F2.L2.sin θ2 + F3.L3.sin θ3

τc = (- 5).(8/6).sin 53° + (0,4).(4/6).sin 90° + (4,8).(8/6).sin 90°

τc = (- 5).(8/6).(4/5) + (0,4).(4/6).(1) + (4,8).(8/6).(1)

τc = - 32/6 + 1,6/6 + 38,4/6

τc = (- 32 + 1,6 + 38,4) / 6

τc = (8) / 6

τc = 4/3

τc = 1,33 N.m

Jawaban : Tidak Ada
karena Data yang diberikan Ambigu antara batang Tongkat diketahui 4 m dan jarak titik AB = BC = CD = DE = 1 m

Note :
Pembahasan saya di atas di asumsikan jika batang panjang nya 4 meter dengan dibagi 6 ruas


atau biar ada jawabannya sesuai denga pilihannya, maka soal di atas saya koreksi dengan MENGHILANGKAN ATAU MENGHAPUS KALIMAT "panjang batang 4 meter".
 
Pakai aja jarak antara titik tadi, AB = BC = CD = DE = 1 m
sehingga jarak L nya berubah menjadi : 
L1 = AC = 2 m (di sebelah kiri titik poros C)
sudut, θ1 = 53°
L2 = BC = 1 m (di sebelah kiri titik poros C)
sudut, θ2 = 90°
L3 = EC = 2 m (di sebelah kanan titik poros C)
sudut, θ3 = 90°
 
Maka : 

τc = F1.L1.sin θ1 + F2.L2.sin θ2 + F3.L3.sin θ3

τc = (- 5).(2).sin 53° + (0,4).(1).sin 90° + (4,8).(2).sin 90°

τc = (- 5).(2).(4/5) + (0,4).(1).(1) + (4,8).(2).(1)

τc = - 8 + 0,4 + 9,6

τc = - 8 + (0,4 + 9,6)

τc = - 8 + 10

τc = 2 N.m

32. yang no 5 help me fisika sma kls 10 smt 2


* Mencari kosntanta pegas
Apabila digantung di pegas k₃, maka bertambah panjang 4 cm
Δx = 4 cm = 0,04 m

F = k • Δx
m • g = k • 0,04
m • 10 = k • 0,04
10m = 0,04k
k = 250m N/m

* Mencari konstanta pegas penganti susunan paralel
kp = k₁ + k₂
kp = 250m + 250m
kp = 500m

* Mencari konstanta pegas sistem
1/ks = 1/kp + 1/k₃
1/ks = 1/500m + 1/250m
1/ks = 1/500m + 2/500m
1/ks = 3/500m
ks = 500m/3 N/m

* Mencari pertambahan panjang
F = k • Δx
m • g = ks • Δx
m • 10 = 500m/3 • Δx
10 = 500/3 • Δx
Δx = 10/(500/3)
Δx = 30/500
Δx = 0,06 m
Δx = 6 cm

Jadi, pertambahan panjang susunan pegas adalah 6 cm
