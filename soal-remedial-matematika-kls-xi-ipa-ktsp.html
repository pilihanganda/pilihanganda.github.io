induksi

p(k+1) =  (k+1)(k+1+1) = (k+1)(k+2)

k(k+1) + 2(k+1) =  (k+1)(k+2)
(k+1)(k+2) = (k+1)(k+2)
ruas kiri = ruas kanan (terbukti)

n(k+1) = k+1

[tex]\sf p(k+1) = \frac{1}{2}(k+1)(k+1+1) = \frac{1}{2}(k+1)(k+2)[/tex]

p(k) +n(k+1) = p(k+1)
