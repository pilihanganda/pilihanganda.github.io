5. soal dari buku detik-detik UN matematika SMA IPA kategori : trigonometri soal nomor 28


Kelas            : 11
Mapel           : Matematika
Kategori      : Trigonometri lanjut
Kata kunci  : trigonometri, aturan cosinus
Kode            : 11.2.3 (Kelas 11 Matematika Bab 3-Trigonometri lanjut)

Gunakan aturan cos untuk segitiga ABD:
BD²=AB²+AD²-2.AB.AD. cos A

Gunakan aturan cos untuk segitiga BCD:
BD²=BC²+CD²-2.BC.CD. cos C

Berdasarkan sifat segiempat tali busur, sudut A dan C berjumlah 180° (karena berhadapan)

BD²=BD²
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos C
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos (180°-A)
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²+2.BC.CD. cos A
AB²+AD²-BC²-CD² = 2.BC.CD. cos A+2.AB.AD. cos A
AB²+AD²-BC²-CD² = cos A (2.BC.CD+2.AB.AD)
cos A = (AB²+AD²-BC²-CD²) / (2.BC.CD+2.AB.AD)
cos A = (4²+6²-3²-3²) / (2.3.3+2.4.6)
cos A = (16+36-9-9) / (18+48)
cos A = 34/66
cos A= 17/33

Jadi, cos ∠BAD adalah 17/33

Jawaban : C

48 = 20 + 23 + 10 - x + x - x

48 = 53 - x

x = 53 - 48

x = 5

27. tolong bikin 1 soal matematika tentang balok kelas 10 SMA


Bab Ruang Dimensi Tiga
Matematika SMA Kelas X


[1] SOAL MENGENAI JARAK

Diketahui balok ABCD.EFGH dengan panjang AB = 6 cm, BC = 8 cm, dan BF = 12 cm. Perpotongan garis EG dengan FH adalah titik O. Perpotongan garis AC dengan BD adalah titik P.
Hitunglah jarak antara titik A dengan garis OC. 

Penyelesaian:

Step-1
Panjang garis AC = √ [AB² + BC²]
AC = √ [6² + 8²]
AC = 10 cm

Panjang garis OC = √ [OG² + CG²]
OC = √ [5² + 12²]
OC = 13 cm

Step-2
Misalkan jarak antara titik A dengan garis OC adalah d.

Pandang ΔAOC
Gunakan rumus sifat kongruensi luas segitiga
⇔ alas₁ x tinggi₁ = alas₂ x tinggi₂
⇔ AC x PO = CO x d
⇔ 10 x 12 = 13 x d
⇔ d = ¹²⁰/₁₃ atau 9 ³/₁₃

∴ Jarak antara titik A dengan garis OC adalah ¹²⁰/₁₃ atau 9 ³/₁₃ cm


[2] SOAL MENGENAI SUDUT

Diketahui balok ABCD.EFGH dengan alas persegi yang memiliki rusuk 8 cm serta tinggi balok 2√17 cm. 
Jika besar sudut antara diagonal ruang AG dengan bidang alas ABCD adalah α, maka cos α adalah ...

Penyelesaian

Step-1
AC = 8√2 cm
AG = √ [AC² + CG²]
      = √ [(8√2)² + (2√17)²]
      = √ [128 + 68]
      = 14 cm

Step-2
∠CAG = α, yakni sudut yang dibentuk oleh garis AG dan garis AC, sebab AC merupakan proyeksi AG pada bidang ABCD. Singkatnya, garis AC mewakili bidang ABCD.

∴ Cos α = AC / AG
             = [8√2] / [14]
             = ⁶/₇ √2

• Untuk x < 0

-(x - 3) = 5

-x + 3 = 5

-x = 2

x = -2

Hp = { -2, 8 }

<=>

Misal |3x + 5| = x

x² + x - 2 = 0

( x - 1 ) ( x + 2 ) = 0

x = 1 atau x = -2

a. untuk x = 1

|3x + 5| = 1

(3x + 5)² = 1²

(3x + 5)² - 1² = 0

(3x + 5 - 1)(3x + 5 + 1) = 0

(3x + 4)(3x + 6) = 0

x = -4/3 atau x = -2

b. untuk x = -2

Tdk ditemukan himpunan karena nilai mutlak tdk mungkin negatif

Hp = { -2, -4/3 }
