12. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

15. ips tentang pembabakan di masa pra aksara jelaskan


1. Zaman Batu
    Terdiri dari 4 zaman:
    a. Palaeolithikum, adalah zaman batu tua. Diperkirakan berlangsung kurang lebih 600.000 tahun silam. Pada zaman ini manusia masih hidup berpindah-pindah (nomaden) dan masih berburu. Alat-alat yang dihasilkan masih dari batu yang kasar.
    b. Mesolithikum, adalah zaman batu tengah, manusia sudah hidup menetap dan masih berburu, alat yang dibuat sudah lebih halus dari zaman sebelumnya.
     c. Neolithikum, adalah zaman batu baru, manusia sudah hidup menetap dan bercocok tanam, alat yang digunakan sudah halus dan serbaguna.
     d. Megalithik, adalah zaman batu besar, manusia sudah dapat membuat bangunan dari batu yang berukuran besar.

2. Zaman Logam
    Manusia sudah dapat membuat perhiasan dan alat-alat lainnya dari logam serta bercocok tanam dan dapat membuat bangunan yang lebih mewah.

27. IPS kehidupan masa pra aksara


Praaksara terdiri dari 2 kata, yaitu pra dan aksara. Pra = sebelum dan aksara = tuisan. Jadi, zaman praaksara adalah zaman ketika kehidupan manusia belum terdapat keterangan tertulis. Pada, kehidupan ini manusia belum meninggalkan keterangan berupa tulisan, melainkan manusia meninggalkan sisa-sisa tulang belulang atau sisa tumbuhan (Fosil) dan benda-benda yang mereka tinggalkan (Artefak).
