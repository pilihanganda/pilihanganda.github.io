24. kakk mau nanya nih , placement test untuk IPA/IPS di SMA it kayak gimana ya test nya ? terus belajar nya apa aja ? makasiii kak


Tes penempatan atau placement test IPA/IPS ketika masuk SMA negeri adalah akumulasi nilai matematika dan IPA dari hasil UN/ASPD dan STTB, serta wawancara. Untuk placement test IPA/IPS untuk SMA swasta adalah tes mengerjakan soal Matematika dan IPA dan wawancara.

Pembahasan

Tes penempatan placement test jurusan IPA/IPS ketika masuk SMA biasanya tergantung pada kebijakan masing-masing sekolah. Begitu pula kebijakan pada SMA negeri dan SMA swasta.

Pada SMA negeri, kebijakan untuk mengadakan tes selain PPDB dengan berbagai jalur (jalur zonasi, jalur prestasi, jalur mutasi orang tua, dan lain-lain), sebenarnya tidak diperbolehkan oleh Dinas Pendidikan. Sehingga, di beberapa SMA negeri hanya memberikan blangko mengenai minat siswa di jurusan IPA/IPS saja. Setelah itu guru menganalisis nilai siswa dari nilai UN/ASPD dan STTB. Jika banyak siswa dengan nilai Matematika dan IPA tinggi, maka sekolah mengadakan tes untuk menentukan rangking siswa. Mata pelajaran yang diujikan adalah Matematika dan IPA dari kelas IX. Begitu pula ketika banyak siswa dengan minat IPS tinggi tetapi minat IPA rendah, maka sekolah akan melakukan hal yang sama.

Sedangkan di SMA swasta, kebijakan melakukan placement test IPA/IPS dapat dilakukan pada awal pendaftaran siswa baru. Tujuan tes ini adalah untuk menyaring siswa yang benar-benar memiliki kemampuan yang baik sesuai dengan jurusannya. Untuk bahan tes sama adalah mata pelajaran Matematika dan IPA yang diperoleh pada saat masih duduk di SMP.

31. Soal no 510 Umptn/b/1995


f(x) = x^3 + 3x^2 + 5 akan turun jika f'(x) < 0
f'(x) = 3x^2 + 6x < 0
=> 3x(x + 2) < 0
x = 0 atau x = -2
+++ (-2) --- (0) +++
-2 < x < 0Kelas : XI (2 SMA)
Materi : Turunan
Kata Kunci : fungsi turun, penyelesaian

Pembahasan :
Diketahui fungsi kuadrat f(x) = x³ + 3x² + 5
Agar fungsi f(x) merupakan fungsi turun, maka f'(x) < 0.
Sehingga
3x² + 6x < 0

Untuk menentukan penyelesaian dari pertidaksamaan kuadrat, maka kita ubah dahulu menjadi persamaan kuadrat
3x² + 6x = 0
⇔ x² + 2x = 0
⇔ x(x + 2) = 0
⇔ x = 0 V x + 2 = 0
⇔ x = 0 V x = -2
Perhatikan gambar terlampir.
Daerah yang negatif merupakan daerah fungsi yang turun, yaitu
-2 < x < 0.
