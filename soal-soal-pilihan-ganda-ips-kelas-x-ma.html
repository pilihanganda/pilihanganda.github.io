4. |2x+3|>|x+1| jawabannya dong MA kelas X


Diketahui:

|2x+3| > |x+1|

Ditanyakan:

x

Jawab:

|2x+3| > |x+1|

|2x+3|² > |x+1|²

4x² + 12x + 9 > x² + 2x + 1

3x² + 10x + 8 > 0

Cari batas

3x² + 10x + 8 = 0

3x² + 6x + 4x + 8 = 0

3x (x+2) + 4 (x+2) = 0

(3x+4)(x+2) = 0

x = -4/3 atau x = -2

Untuk menentukan intervalnya, subtitusikan bilangan antara -4/3 dan -2 ke persamaan.

Misal x = -5/3.

3(-5/3)² + 8(-5/3) + 8 ... 0

3(25/9) - (40/3) + 8 ... 0

75 - 120 + 72 ... 0

27 > 0

(Memenuhi)

Maka, -2 < x < -4/3.

6. soal :Ada 5 siswa kelas X, 3 siswa kelas XI, dan 4 siswa kelas XI. Mereka duduk di kantin mengelilingi meja bundar. Dengan berapa cara mereka dapat duduk jika siswa yang kelasnya sama selalu duduk berdampingan? Jawab: Belajar Praktis Matematika SMA/MA Kelas Xll​


ada 3 unsur (n) yaitu siswa kelas X, siswa kelas XI, siswa kelas XII,

yg bersyarat :

siswa kelas X harus selalu duduk berdampingan (5!)

siswa kelas XI harus selalu duduk berdampingan (3!)

siswa kelas XII harus selalu duduk berdampingan (4!)

permutasi siklis (n-1)! = (3-1)! = 2!

berapa banyak cara mereka duduk

= 2!×5!×3!×4!

= (2×1)×(5×4×3×2×1)×(3×2×1)×(4×3×2×1)

= 2×120×6×24

= 34560

jadi, ada 34.560 cara
