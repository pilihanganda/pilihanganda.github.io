Jawaban:

1. 8x + 8y - 11z

2. 20x - 4y - 8z

Penjelasan dengan langkah-langkah:

1.

PERTANYAAN:

Hasil sederhana dari: 2x + 3y + 5y - 4z + 6x - 7z

JAWABAN:

= 2x + 3y + 5y - 4z + 6x - 7z

= 2x + 6x + 3y + 5y - 4z - 7z

Masing-masing:

2x + 6x = 8x

3y + 5y = 8y

-4z - 7z = -11z

= 8x + 8y -11z

2.

PERTANYAAN:

Hasil sederhana dari: 5x - 4y + 9x - 8z + 6x

JAWABAN:

= 5x - 4y + 9x - 8z + 6x

= 5x + 9x + 6x - 4y - 8z

Masing-masing:

5x + 9x + 6x = 20x

= 20x - 4y - 8z
