2) volume = luas alas x t

   172.480 = luas alas x 70

   luas alas = 172.480 : 70

                  = 2.464 cm²

3) volume = luas alas x t

   100.480 = 1.256 x t

                t = 100.480 : 1.256

                  = 80 cm

4) diameter = 0,84 m

   jari-jari = diameter : 2 = 0,84 : 2 = 0,42 m

27. tolong bantu jawab soal di LKS​


Jawab:

Penjelasan dengan langkah-langkah:

x = proyeksi tiik di pusat sisi CDHG

y = proyeksi titik di 1/2 DF

Hx = 1/2 x 6√2

     = 3√2 cm

xy = 1/2 x 6

   = 3 cm

Hy = √(Hx² + xy²)

     = √((3√2)² + 3²)

     = √((9 x 2) + 9)

     = √(18 + 9)

     = √27

     = 3√3

Karena tegak lurus dengan titik H

Maka (6√3)/2 = 3√3
