30. jika OPS adalah garis lurus, maka besar sudut QPR adalah?​


Jawaban:

<OPS = garis pelurus = 180°

maka,

<OPQ + < QPR + <RPS = <OPS

20° + (10x - 2)° + (3x + 6)° = 180°

13x° + 24 = 180°

13x = 180° - 24°

13x = 156°

x = 12°

subsitusikan ke

<QPR~ = 10x - 2 = 10(12)° - 2 = 120° - 2 = 118°

Jadi, besar sudut QPR adalah 118°
